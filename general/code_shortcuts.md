# Command Shortcuts
  ### Jupyter shorcuts for VS code
    - `Enter` : Go into edit mode
    - `Esc`: Switch to command mode
    - 
    - `A`: Add a cell above
    - `B`: Add a cell below
    - `Alt+up`: move a cell up
    - `Alt+down`: move a cell down
    - `dd`: Delete a selected cell
    - `M`: Switch the cell to Markdown
